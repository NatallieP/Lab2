Luego nos hablo sobre que era ELEMENTO dentro del HTML y la importancia de este. Ya que todo los cambios que queramos hacer se hace ahí.

¿QUE ES UN ELEMENTO EN HTML?
Es un bloque básico de construcción que define la estructura y el contenido de una página web. Cada elemento HTML está compuesto por una etiqueta de apertura, el contenido (opcional)
y una etiqueta de cierre (en la mayoría de los casos).

NO TODOS LOS ELEMENTOS CUENTAN CON ETIQUETA DE CIERRE

BR, Es una etiqueta que se utiliza para insertar un salto de línea. A diferencia de un párrafo "p", que crea un bloque de texto separado, el "br" solo mueve el texto a una nueva línea sin agregar espacio adicional.

ESTRUCTURA BÁSICA DE HTML
Define cómo se organizan y estructuran los diferentes elementos de contenido en una página web. Esta estructura sigue un conjunto de etiquetas que son esenciales para garantizar que el
documento sea interpretado correctamente por los navegadores web.
